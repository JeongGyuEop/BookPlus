## 🛠️ 사용 기술 및 환경

### 백엔드

- **언어:** Java (Servlet, JSP)
- **데이터베이스 연동:** JDBC

### 프론트엔드

- **기술:** HTML, CSS, JavaScript, jQuery

### 데이터베이스

- **DBMS:** MySQL

### 서버

- **웹 서버:** Apache Tomcat

### 라이브러리

- Gson
- JSoup
- JSON-Simple
- MySQL Connector

### 개발 및 협업 환경

- **IDE:** Eclipse
- **버전 관리:** Git, GitHub
- **문서 관리:** Notion
